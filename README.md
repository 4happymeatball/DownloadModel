# DownloadModel
下载器探索

使用方法：

Add it in your root build.gradle at the end of repositories:
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
Add the dependency:
```
  dependencies {
	        implementation 'com.github.renzhenming:DownloadModel:v1.0.2'
	}
```
